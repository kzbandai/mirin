# mirin
- Updates manager by Go

```
USAGE:
   mirin [global options] command [command options] [arguments...]

VERSION:
   0.0.1

AUTHOR:
   kzbandai

COMMANDS:
     anyenv    update
     brew      upgrade
     composer  self-update
     gcloud    components update
     npm       install -g npm@latest

GLOBAL OPTIONS:
   --version, -v  print the version

COPYRIGHT:
   MIT
```
